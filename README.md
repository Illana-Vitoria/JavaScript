JavaScript Essentials - IFTM 🚀

Repositório dedicado aos estudos de JavaScript (ES6+) e lógica de programação, desenvolvidos durante o curso de Tecnologia em Sistemas para Internet no IFTM - Campus Uberlândia Centro.

📋 Sobre o Repositório

Este repositório documenta meu aprendizado em tornar interfaces web interativas e funcionais. O foco aqui vai além da sintaxe, explorando a resolução de problemas reais e a integração com o DOM e APIs.

🛠 Conceitos Explorados

Lógica de Programação: Estruturas de repetição, condicionais e tratamento de exceções.

ES6+: Uso de arrow functions, destructuring, spread/rest operators e módulos.

Manipulação de DOM: Criação dinâmica de elementos, eventos de formulário e alteração de estilos em tempo real.

Assincronismo: Consumo de dados via Fetch API, uso de async/await e manipulação de arquivos JSON.

📂 Destaques de Estudo

Validação de Formulários: Scripts para garantir a integridade dos dados inseridos pelo usuário.

Integração com Backend: Exemplos de consumo de APIs (como a API do GitHub) para exibição de informações dinâmicas.

Projetos Práticos: Implementações como o sistema de críticas de cinema ("Como se Lhama?") e lógica de autenticação segura.
